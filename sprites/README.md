## 图片规格要求
- **尺寸**: 128×128 像素 (正方形)
- **格式**: PNG 透明背景
- **风格**: 像素艺术或卡通风格
- **内容**: 精灵正面或侧面立绘

## 使用说明
1. 将精灵立绘图片按ID命名放入此文件夹
2. 游戏会自动加载对应图片替换默认emoji图标
3. 如果图片缺失，游戏会显示默认emoji图标

## 制作指南
1. 可以使用像素艺术工具如 Aseprite、Piskel
2. 也可以使用AI生成工具创建
3. 确保背景透明，精灵主体清晰
4. 保持风格统一

## 游戏加载逻辑
```javascript
// 游戏会尝试加载本地图片
const spritePath = `sprites/${pokemon.id}.png`;
// 如果图片存在则显示，否则使用emoji图标
```
