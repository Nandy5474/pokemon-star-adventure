# 星辉传说 - 宝可梦风格冒险游戏

一个基于 HTML5 Canvas 的宝可梦风格冒险游戏，支持桌面和移动端。

## 游戏特色
- 🎮 完整的精灵捕捉、对战、进化系统
- 🗺️ 多张地图探索（起始镇、常盘森林、港口镇、阳光海滩、矿山镇、传说洞穴）
- 🏆 道馆挑战与徽章收集
- 📱 响应式设计，支持手机和电脑
- 💾 本地存档功能
- 🎨 可自定义精灵立绘

## 在线游玩
访问 [https://nandy5474.github.io/pokemon-star-adventure/](https://nandy5474.github.io/pokemon-star-adventure/)

## 本地运行
直接双击 `index.html` 或在浏览器中打开即可。

## 精灵立绘
将 128×128 像素的 PNG 图片放入 `sprites/` 文件夹，按精灵 ID 命名（如 `001.png`），游戏会自动加载。

## 技术栈
- HTML5 Canvas
- 原生 JavaScript
- CSS3 动画
- 本地存储
